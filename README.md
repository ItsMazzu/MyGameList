# MyGameList

Projeto inspirado no MyAnimeList, porém focado em jogos.  
Permite que o usuário liste seus jogos (favoritos, jogando, ou planeja jogar) e avalie títulos, deixando essas avaliações disponíveis para toda a comunidade.

## Tecnologias Utilizadas

- React.JS
- Node.JS  
- HTML  
- CSS

## Estrutura de Branches

- `main` → branch final do projeto
- cada dev cria e trabalha na sua própria branch
- ao final, merge de todas as branches de dev → `main`
